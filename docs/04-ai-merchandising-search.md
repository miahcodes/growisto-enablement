# AI-Powered Merchandising & Search

## Introduction

Search and product discovery are the **highest-leverage conversion opportunities** in e-commerce. Industry data shows:

- **35-40%** of e-commerce sessions start with search
- **Search users convert 2-3x higher** than browse-only users
- **Poor search = immediate bounce**: 68% of users leave after one failed search

For Indian enterprise merchants with 1,000-50,000+ SKUs, traditional keyword-based search fails spectacularly. Customers search for "kurta for wedding" and get no results because products are tagged "ethnic wear" or "festive kurta." Revenue is lost every day.

**Shopify's AI-powered search and merchandising** solve this through semantic understanding, intent recognition, and personalized recommendations.

## Semantic Search: AI That Understands Intent

### What is Semantic Search?

**Traditional keyword search**:
```
Customer searches: "red dress for party"
System looks for: exact matches of "red" AND "dress" AND "party" in product titles/descriptions
Misses: "Crimson cocktail gown", "Ruby evening dress", "Scarlet formal wear"
Result: Incomplete results, lost sales
```

**Semantic search (AI-powered)**:
```
Customer searches: "red dress for party"
AI understands:
- Color: red, crimson, ruby, scarlet, burgundy, wine
- Product: dress, gown, evening wear
- Occasion: party, cocktail, formal, celebration
- Intent: Looking for dressy, formal options

Returns: All relevant products regardless of exact keyword matches
Result: Comprehensive results, higher conversion
```

### How Shopify's Semantic Search Works

#### 1. **Natural Language Understanding**
AI processes search queries like a human would:

| **Customer Query** | **AI Interpretation** | **Products Returned** |
|--------------------|----------------------|----------------------|
| "something for Diwali" | Occasion: Diwali, festive wear | Ethnic wear, traditional clothing, festive decor |
| "comfortable work shoes" | Category: shoes, Use case: office, Attribute: comfort | Formal shoes with comfort features, loafers, low heels |
| "gift for mom under 2000" | Intent: gifting, Recipient: mother, Budget: ₹2000 | Gift-appropriate items <₹2000, sorted by popularity |
| "मेरी शादी के लिए कुछ" (Hindi: "something for my wedding") | Language: Hindi, Occasion: wedding, Role: bride/groom | Wedding apparel, accessories, ceremonial items |

#### 2. **Synonym & Contextual Understanding**

AI automatically understands:

**Fashion synonyms**:
- Kurta = kurti = tunic = ethnic top
- Saree = sari = traditional wear
- Lehenga = lengha = ghagra = chaniya choli

**Regional variations**:
- Chole bhature = chola bhatura
- Dosa = dosai = dose
- Kurta = kurtha = kurtha

**Misspellings**:
- "niroos" → Nike shoes
- "adiddas" → Adidas
- "leevis" → Levis

#### 3. **Attribute Extraction**

AI extracts multiple attributes from a single query:

```
Search: "blue cotton kurta for men size L under 1500"

AI extracts:
- Color: blue
- Material: cotton
- Product type: kurta
- Gender: men
- Size: L
- Price: <₹1,500

Filters applied automatically (no manual filtering needed)
```

## Smart Collections & Auto-Categorization

### What Are Smart Collections?

Smart Collections automatically populate based on rules (tags, price, inventory, etc.). AI enhances this with:
- Automatic tagging and categorization
- Trend detection
- Seasonal adjustments

**Challenge**: Merchant adds 500 new products monthly, manual categorization takes 3-4 hours

**AI Solution**:
```
Upload Product: "Women's Floral Print A-line Midi Dress - Summer Collection"

- Image: Floral pattern, A-line silhouette, midi length, light fabric
- Description: Summer, casual, flowy
- Similar products: Past categorization patterns

Auto-assigns:
- Primary Category: Women > Dresses > Midi Dresses
- Collections: "Summer Essentials", "Floral Prints", "Casual Wear", "₹1000-₹2000"
- Tags: floral, a-line, midi, summer, casual, cotton-blend

- Occasion: Casual, Day wear

Time saved: 3 hours → 30 seconds (99% reduction)
Accuracy: 94% (requires manual adjustment on 6% of products)
```

## Best Practices for Implementation

### 1. **Optimize Product Data for AI**

**High-quality inputs = better AI outputs**

**Product Title Best Practices**:
```
❌ Poor: "Beautiful Dress"
✓ Good: "Women's Floral Maxi Dress - A-line, Summer, Cotton"

❌ Poor: "AMDRS234" (SKU as title)
✓ Good: "Nike Air Zoom Pegasus 40 Running Shoes - Men's Black/White"

Why: AI uses titles for semantic understanding; descriptive titles = better search results
```

**Product Description Best Practices**:
```
Include:
- Material: "100% cotton", "silk blend", "breathable mesh"
- Fit: "Regular fit", "slim fit", "oversized"
- Occasion: "office wear", "party wear", "casual"
- Care instructions: "machine washable", "dry clean only"
- Use case: "ideal for summer", "perfect for weddings"

Why: AI extracts attributes from descriptions to power search and recommendations
```

**Image Best Practices**:
```
- High resolution: Minimum 1000x1000px
- Clean background: White or neutral (AI analyzes product better)
- Multiple angles: Front, back, detail shots
- Lifestyle images: On models/in use (helps visual search)
- Consistent lighting: Enables accurate color detection

Why: Visual search and auto-categorization depend on image quality
```

### 2. **Monitor Search Analytics**

**Key metrics to track weekly**:

```
Search Performance Dashboard:

1. Zero-Result Searches
   - Goal: <5%
   - Action if >5%: Review failed queries, add synonyms, adjust categorization

2. Search-to-Cart Rate
   - Benchmark: 15-25%
   - Action if low: Check search relevance, review top queries

3. Search-to-Purchase Rate
   - Benchmark: 6-10%
   - Action if low: Audit product pages of top search results

4. Top Searched Terms (No Results)
   - Review: Products you should add to catalog
   - Example: "plus size kurta" getting searches but you have no plus-size products = opportunity

5. Search Revenue Contribution
   - Benchmark: 30-45% of total revenue
   - Track trend: Should increase as AI improves
```
